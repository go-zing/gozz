# 插件

Gozz 提供了一系列强大的内置插件，与其说 Gozz 提供了这些内置插件， 不如说为了整合这些插件，给它们做了一个统一交互叫 Gozz。

  • Wire 自动化依赖注入 及 AOP代理
  • Api 基于 interface 快速提供 API接口
  • Impl 同步 interface 和 implement
  • Doc 基于注释生成运行时文档
  • Tag 模版化管理结构体字段标签
  • Orm 生成数据实体映射代码
  • Option 生成 Funcitonal Options 风格代码
